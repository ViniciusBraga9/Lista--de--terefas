# 📝 Lista de Tarefas em Python

Um projeto simples de lista de tarefas feito em Python para treinar manipulação de listas, loops e condicionais.

## 🚀 Funcionalidades
- Listar tarefas
- Adicionar tarefa
- Pesquisar tarefa pelo código
- Remover tarefa
- Substituir tarefa
- Avaliar o programa com nota
- Encerrar o programa
